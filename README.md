# [SpeedTyper](http://jonathanbgjo.com/JavaScript-TypingGame/)

SpeedTyper is a game where you type as many words as you can within the time remaining! 1 word completed is 1 score.

* Technologies

  * JavaScript
  * HTML
  * CSS
